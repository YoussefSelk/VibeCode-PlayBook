# Code of Conduct

## Our Standard

We are committed to making participation in this project a respectful, harassment-free experience for everyone.

We expect contributors to:

- be respectful and constructive
- assume good intent while discussing technical decisions
- focus feedback on ideas, code, and outcomes
- avoid personal attacks, insults, discrimination, and harassment

## Unacceptable Behavior

Examples of unacceptable behavior include:

- harassment, intimidation, or hateful language
- personal insults, ad hominem attacks, or threats
- repeated disruptive behavior after moderator guidance
- sharing private information without consent

## Scope

This Code of Conduct applies in all project spaces, including issues, pull requests, discussions, and community channels linked to this repository.

## Reporting

To report abuse, harassment, or serious violations, contact the maintainers privately:

- Open a private security advisory for security-sensitive incidents when appropriate.
- Otherwise use repository owner contact in GitHub profile.

## Enforcement

Maintainers are responsible for clarifying standards and enforcing this policy fairly and consistently.

Possible enforcement actions include:

- warning
- temporary interaction limits
- removal of comments/content
- temporary or permanent ban from project spaces

## Project Values

We value clarity, kindness, and productive collaboration. We prefer public, transparent discussion for normal project work, with private channels reserved for sensitive reports.
