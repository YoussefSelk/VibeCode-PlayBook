# VibeCoders Role Summary

- prompt-engineer: turns rough asks into clear execution prompts
- db: owns schema, migrations, persistence contracts
- backend: owns APIs, business logic, validation, auth
- frontend: owns UI, client behavior, accessibility
- devops: owns CI/CD pipelines, deployments, infra configuration, and operability checks
- tester: validates real runtime behavior across layers
- security: reviews auth, secrets, permissions, sensitive data exposure
- legal: reviews privacy, consent, retention, policy-sensitive flows
- reviewer: final regression and risk gate before done
