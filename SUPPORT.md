# Support

## Where To Ask Questions

For usage questions and help:

- Open a GitHub issue with clear context and reproduction details.

For bugs:

- Use the bug report template.

For ideas:

- Use the feature request template.

## Before Opening An Issue

Please check:

- README
- CONTRIBUTING
- existing open and closed issues

## What To Include

- expected behavior
- actual behavior
- reproduction steps
- environment details (OS, shell, versions)
- logs or screenshots when relevant

## Response Expectations

This is an open source project maintained with best effort.
Polite, precise reports are reviewed faster and lead to better outcomes.
